연산자

수치 연산자(산술)
 + - x / %

 /는 수학과 다르다
  3.0/2=1.5
  3/2=1
  정수/정수 = 정수 (실수는 취급하지않는다 소수점이 버려짐,정확도 떨어짐)
  실수/정수,실수/실수 = 실수
  정확한 결과를 얻으려면 자료형 변환으로 한쪽을 실수로 만들어야함(계산식에 프린에x)

 %는 나머지 연산자, 홀수/짝수 판별 이용, 배수,약수를 구할때 이용
  나누었을 때 나머지라는 표현이 들어가면 % 연산자
  5%2 = 1
  9%7 = 2
 
 축약된 형태의 연산자(p.65)-결합연산자
  a=a?b = a?=b
  a=a+b; = a+=b;
  덧셈,곱셈제외 연산순서가 바뀌면 사용불가

여러 변수를 선언하는 방법
 자료형 변수명1, 변수명2; (자료형이 같으면 ,로 여러개 쓸수있다)
 자료형 변수명5, 변수명 6 = 초기값, 변수명7, 변수명8 = 초기값; (초기값 넣고안넣고 가능)

할당연산자(대입연산자)
 '='
 수학에서 a+1=b, b=a+1이 같은의미
 프로그래밍언어에선 a+1=b가 불가능
 오른쪽에 있는 값을 왼쪽 변수에 저장한다고 기억하기(덮어쓰기)(핵심)
 =과 ==을 구별해야함

증강연산자(p.67)
 ++,--
 ++: 최종적으로 변수가 1 증가
 --: 최종적으로 변수가 1 감소
 전위형 증강 연산자:++변수명, --변수명
  변수의 값이 먼저 증가하고 동작
 후위형 증강 연산자:변수명++, 변수명--
  동작을 하고 변수의 값이 증가 또는 감소

전위형과 후위형이 어려우면 증감과 동작을 같이하지말고 따로하라

논리 연산자(p.72)
 결과값이 참또는 거짓이 나오는 연산자
 &&:and연산자. ~하고,~이고,
  둘다 참이면 참. 나머지는 거짓
   a학점은 성적이 100점이하,90점 이상이다
   내성적이 85이면 거짓 100점이상은 참이나 90점이상이 거짓이므로
 ||:or 연산자. ~하거나, 또는
  둘다 거짓이면 거짓, 나머지는 참(하나라도 참이면 참)
   토익을 보려면 신분증이나 면허증이 있어야한다
   둘다 없으면 응시가능? =불가
   민증만 갖고오면 가능? = 가능
 !:not 연산자. 반대
  성별이 남자와 여자중 남자의 반대는 여자, 여자의 반대는 남자

 &&,|| 연산자 양 옆에는 조건식이 와야함
 조건식은 참 또는 거짓으로 판별할 수 있는 식

관계연산자(비교연산자, P.71)
 두 숫자(정수, 실수)의 크기를 비교하는 연산자
 결과가 참 또는 거짓
 <,>,<=,>=,==,!=
 =>,=<라는 기호는 없다
 =와 ==를 구분할 수 있어야한다

비트연산자(p.68)
 &,|,~,^
 비트연산자는 연산 결과가 특정 값이다. 참거짓이 아님
 비트 연산자는 정수 또는 문자 변수에만 사용 가능
 변수를 비트로 나열한 후, 각 비트별로 논리 연산을 함, 1을참 0을 거짓

자료형변환(p.75)
 변수나 상수의 자료형을 일시적으로 바꿔줌
 변환시키고자 하는 변수앞에(자료형)
 ^:xor,배타논리 연산자, 서로다르면1, 같으면0
 LED 제어할 때 사용

비트 시프트 연산자 (p.70)
 변수를 비트로 나열한 후 이동시키는 연산자
 >> 오른쪽으로 이동. 2의n제곱으로 나눈 효과
 << 왼. 2의n제곱으로 곱한 효과

조건선택연산자
 조건식 ? 참일때 값: 거짓일때 값:
 나중에 조건식으로 대체가능
 조건문을 쓰는데 간결하게 쓸때 많이 사용
